# epp

Electron Application

- electron
- webpack
- react
- bootstrap
- electron-app


## Dev

```
$ npm install
$ npm run start-hot
$ npm run start-dev
```

### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
